# 📚 Data Structures Using C (DSU) - Diploma Programs

Welcome to my **Data Structures Using C (DSU)** repository! 🚀

This repository contains C programs based on the **MSBTE Diploma Computer Engineering/Computer Technology** syllabus. It is designed to help students understand the fundamentals of data structures through simple, well-structured programs.

## 🎯 Objective

The goal of this repository is to:
- Practice Data Structures using C.
- Help diploma students prepare for practicals and examinations.
- Provide clean and easy-to-understand code.
- Build a strong programming foundation.
